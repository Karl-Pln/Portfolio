Cette photo présente ici la classe qui contient les informations de la base de donnée SQL. elle permet la connexion a l'aide de la fonction getconnexion().

Cette photo présente le fichier "Index.php". Ce dernier permet le bon fonctionnement du site en reliant la base de donnée et les controleurs grâce aux "requires"

Comme la photo précedente, cette photo montre un exemple de requete SQL pour gérer les données. Ici la requete permet de supprimer les données grâce à "delete"

Update :

Tout comme les photos précedentes, cette photo montre un exemple de requete SQL. Cette fois-ci, la requete permet de mettre à jour les données grâce à "update"

Controleur :

Cette photo montre un exemple de méthode que l'on peut trouver dans un controleur. Ici, la méthode permet de récupérer les données d'un formulaire et de les envoyer au modèle pour traitement.

Controleur Principal:

Le role du controleur principal est d'effectuer des actions telle que "connexion". Cette action est définie par le "case". pour chaque case, il va chercher dans le controleur la methode voulue

Vue :

Le rôle de la vue est l'affichage des données de l'utilisateur. Ici, la vue récupère les données des vehicules "get" pour les afficher sur la page.
