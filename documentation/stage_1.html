Documentation du Projet : Site HTML

Présentation du projet

chargement des images :

Ce script C# est un extrait de la methode LoadImage qui permet de charger les images afin de les afficher.

Affichage de l'image suivante :

Cette image est un Extrait de la méthode ShowNextImage.
Elle permet d'afficher les images suivantes parmi la liste d'image préalablement chargées.

Transition d'image :

Ce script est un extrait de la méthode Transitions qui permet d'avoir une transition fluide entre chaque images.

Mise a jour du dossier :

Ce script issu de la méthode refresh_folderLocal permet de rafraichir la liste d'image présente dans dossier en allant regarder le dossier partagé.
